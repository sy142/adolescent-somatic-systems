# R

Analysis and figure scripts. Scripts use a configurable data path; set it to your local microdata file before running.

- `analiz.R` — main analysis (measurement models, regressions, network, ML).
- `figures.R` — figure generation.
- `revizyon_analizleri.R` — sensitivity and robustness analyses for the revision (latent MIMIC, continuous outcomes, alternative categorizations, weighted models, multiple imputation, bifactor and bifactor MIMIC).
